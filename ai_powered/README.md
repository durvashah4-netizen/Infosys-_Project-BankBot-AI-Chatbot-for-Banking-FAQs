# AI Powered Banking Chatbot

## Tech Stack
- Python
- LangChain
- Groq LLM
- Local LLaMA (GGUF)
- SQLite
- Streamlit

## Features
- Intent detection via LLM
- Entity extraction
- Confidence scoring
- Fallback handling
- CLI + UI support

## Run
```bash
pip install -r requirements.txt
streamlit run main_app.py
